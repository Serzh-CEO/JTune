# 🎵 JTune

**JTune** is an offline MP3 player that scans a specified directory for `.mp3` audio files and provides a convenient interface for listening to them. The main programming language is Java, with JavaScript also being used. Although it is a desktop application, its interface is designed in a web-style layout using JavaFX, FXML, HTML, and CSS, making it modern and user-friendly.

---

## 🚀 Features
- 📁 Scan local folders for MP3 files  
- 🎧 Play music offline  
- 🗂️ Simple music library management  
- 💻 Web-style desktop UI  
- ⚡ Fast and lightweight performance  

---

## 🛠️ Technologies Used
- Java (core logic)
- JavaFX (desktop UI framework)
- FXML (UI structure)
- mp3agic (work with mp3 files)
- sqlite (work with db)
- HTML (UI rendering elements)
- CSS (styling)
- JavaScript (additional UI logic)

---

## 📌 Project Idea
JTune was created as a simple and modern offline music player focused on usability and clean design, without requiring internet or external services.

---

## 🖥️ UI Concept
Even though JTune is a desktop application, its interface behaves like a web page, providing a more modern and interactive user experience.
